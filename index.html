Lista de Materiales (BOM)

Todo lo que necesitas para construir tu Simbionte.

Materiales Base (Ambas Versiones)

  • Imanes de Neodimio N35 o N52 (10x2 mm)

    El corazón del sistema. Sirven para anclar magnéticamente el Simbionte a la impresora, fijar las caras intercambiables y añadir accesorios extra. Necesitarás al menos 4 imanes base.

  • Piezas Impresas (PLA o PETG)

    Descarga los STLs desde Printables. Puedes usar PLA sin problemas, ya que el exterior de la Prusa Core One no genera calor suficiente para deformarlo.

Guía de Cableado Eléctrico

Sigue detalladamente este mapa de pines para asegurar una intercomunicación perfecta sin dañar los componentes.

Hub de Distribución de 3.3V (WAGO)

Dado que la placa ESP32 cuenta con salidas limitadas de voltaje, el conector WAGO unifica y distribuye la energía estable de 3.3V de la siguiente manera:

  • Slot 1: Pin 3V3 (ESP32)
  • Slot 2: VIN / VCC (Pantalla OLED)
  • Slot 3: VIN / VCC (Sensor AHT10)
  • Slot 4: VCC (+) (Buzzer Pasivo)
*Nota: Asegúrate de bajar completamente las pestañas del conector WAGO para evitar falsos contactos.

Bajo el Capó (Arquitectura)

El software del Simbionte no es un simple script. Está diseñado bajo principios de ingeniería de software orientada a objetos (OOP) y optimización IoT extrema.

  • C++ Orientado a Objetos

    Motores gráficos y de audio separados en clases independientes (`Renderer`, `Brain`, `Games`) para modularidad total.

  • Compresión GZIP Hexadecimal

    Todo el panel web está minificado y comprimido reduciendo el uso de RAM en un 80%.

  • Memoria EEPROM y Bitmasking

    El sistema de logros utiliza máscaras de bits para guardar el progreso histórico permanentemente.

Inteligencia Reactiva

El código C++ traduce los eventos de PrusaLink en expresiones dinámicas.

^ _ ^

Printing (Imprimiendo)

Estado normal. Muestra porcentaje, temperatura y animaciones autónomas.

X _ X

Error Crítico

El buzzer pasivo emite una alarma SOS musical y los ojos muestran una X.

@ _ @

Mareado (Simulado)

Animación especial que refleja inestabilidad o movimientos agresivos.

* _ *

Job Finished

Celebra con estrellas y actualiza el contador histórico en su memoria.

Flashea en 1 Clic

No necesitas instalar Arduino IDE ni librerías complejas. Conecta tu ESP32 por USB y presiona el botón.

⚠️ Por favor, usa Google Chrome o Microsoft Edge en tu computadora para poder flashear el código.

Pasos post-instalación:

  1. Conecta el ESP32 a la corriente.
  2. Busca en tu teléfono el Wi-Fi "SIMBIONTE_SETUP" (Contraseña: Cocolab3D).
  3. Ingresa tus datos, la IP de tu computador/simulador y elige la unidad térmica (C/F).

¿Ya instalaste el Simbionte y quieres ver su Panel de Control?

Para volver a entrar y modificar parámetros en el futuro, asegúrate de estar conectado al mismo Wi-Fi y abre tu navegador web en la dirección http://simbionte.local o escanea tu red con la app Fing para descubrir su IP.
